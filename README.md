# 🦖 武吉拉 Wujila - 股票分析系統

一個基於 Streamlit 的即時股票分析工具，支援台股與美股查詢。

## 功能特色

- 📊 即時股價查詢（台股/美股）
- 📈 多週期 K 線圖（1分鐘到月線）
- 📉 技術指標分析（MA5/MA10/MA20、KD指標）
- 🎨 美觀的卡片式介面設計
- 📱 響應式佈局，支援手機瀏覽

## 安裝說明

1. 安裝相依套件：
```bash
pip install -r requirements.txt
```

2. 執行應用程式：
```bash
streamlit run app.py
```

## 使用方式

1. 在搜尋框輸入股票代號：
   - 台股：輸入4位數代號（例如：2330）
   - 美股：輸入英文代號（例如：AAPL、NVDA）

2. 選擇時間週期查看不同週期的 K 線圖

3. 查看技術分析報告與操作建議

## 技術指標說明

- **MA5/MA10/MA20**：5日、10日、20日移動平均線
- **KD指標**：隨機指標，用於判斷超買超賣
  - K值 > 80：超買區
  - K值 < 20：超賣區
  - 黃金交叉：K線向上穿越D線（買進訊號）
  - 死亡交叉：K線向下穿越D線（賣出訊號）
